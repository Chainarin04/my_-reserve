2. ใช้คำสั่ง Clone
เปิด Terminal ใน VS Code (หรือ Command Prompt) แล้วทำตามนี้:

ย้ายที่อยู่ไปโฟลเดอร์ที่ต้องการเก็บงาน: (เช่น htdocs ของ XAMPP)

Bash
cd C:\xampp\htdocs
พิมพ์คำสั่ง Clone:

Bash
git clone [วาง URL ที่ก๊อปมาที่นี่]
ตัวอย่างเช่น:

Bash
git clone https://github.com/Chainarin04/my_reservation.git
